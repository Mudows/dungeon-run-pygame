# MSAT Dungeon Run

## Mecânicas
- Equipamento define habilidade
- Herança do Herói: Quando o personagem morre, o progresso da fase é resetado. Porém um item aleatório dele (excluindo consumíveis) é transferido para o personagem novo.
- Progressão se dá por escolhas no fim de cada nível, podendo aumentar o máximo de vida ou mana.
